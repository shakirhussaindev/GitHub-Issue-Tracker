 
    - 1️⃣ What is the difference between var, let, and const?
      Ans: 
      var is a function scoped, it could be re-declare, re-assign and hoisted. Not recomended.
           let is a block scoped, and it could be re-assign, not re-declare, hoisted in TDZ (Temporal Dead Zone)
           const is a block scoped, it's not re-declare, re-assign and not hoisted.


    - 2️⃣ What is the spread operator (...)?
      Ans: 
      Spread operator is a JavaScript feature that expands element of an iterable (like Array, string) object into a new structure.

    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    Ans:
    map() transform each element and returns a new array, 
    filter() returns a new array with elements that match a condition,
    forEach iterate over an array and returns nothing.
